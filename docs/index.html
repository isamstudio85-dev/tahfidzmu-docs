Kebijakan Privasi

TahfidzMU — Aplikasi Tahfidz Mudah untuk Berbagai Hafalan

Terakhir diperbarui: 8 Juli 2026

1. Pendahuluan

TahfidzMU ("kami", "aplikasi") adalah aplikasi Tahfidz Mudah yang dikembangkan oleh iSam Studio untuk memudahkan pengelolaan hafalan secara fleksibel, modern, dan praktis. Aplikasi ini merupakan versi awal yang akan terus dikembangkan sesuai kebutuhan lembaga. TahfidzMU tidak hanya digunakan untuk hafalan Al-Quran, tetapi juga dapat digunakan untuk berbagai jenis hafalan lain. Saat ini TahfidzMU tersedia melalui aplikasi mobile, web admin untuk operasional pesantren, serta panel super admin untuk pengelolaan banyak pesantren.

2. Data yang Kami Kumpulkan

Kami mengumpulkan data yang diberikan oleh pengguna atau administrator pesantren untuk mendukung operasional aplikasi, antara lain:

  • Data Akun: Nama, username, email, nomor telepon, dan jabatan atau peran pengguna.
  • Data Santri: Nama, NIS, kelas, halaqah, nama orang tua atau wali, nomor HP wali, tanggal lahir, dan foto profil.
  • Data Hafalan: Riwayat setoran, murojaah, tasmi, nilai, catatan perkembangan, dan data pendukung hafalan lainnya.
  • Data Pesantren: Nama pesantren, NPSN, alamat, nomor telepon, email institusi, website, dan pimpinan.
  • Data Foto: Foto profil atau foto verifikasi yang diunggah dalam proses operasional aplikasi.

3. Bagaimana Kami Menggunakan Data

Data yang dikumpulkan digunakan untuk:

  • Mengelola berbagai program hafalan, termasuk namun tidak terbatas pada hafalan Al-Quran.
  • Menampilkan progres hafalan kepada admin, musyrif, pengawas, dan orang tua atau wali.
  • Mengelola presensi, kelas, halaqah, wisuda, dan administrasi pesantren lainnya.
  • Mendukung operasional melalui aplikasi mobile, web admin, dan panel super admin sesuai hak akses masing-masing.
  • Melakukan autentikasi, otorisasi, dan pengamanan akses pengguna.

4. Penyimpanan dan Keamanan Data

  • Data disimpan di Google Firebase, termasuk Firestore dan Firebase Storage.
  • Akses data dilindungi dengan kontrol hak akses berbasis peran.
  • Data antar pesantren diisolasi agar tidak saling dapat diakses.
  • Koneksi antara aplikasi dan server menggunakan enkripsi TLS atau SSL.

5. Berbagi Data dengan Pihak Ketiga

Kami tidak menjual, menyewakan, atau membagikan data pribadi pengguna kepada pihak ketiga untuk tujuan komersial. Data hanya dapat diakses oleh pengguna yang berwenang, tim teknis untuk pemeliharaan sistem, dan penyedia infrastruktur cloud yang relevan.

6. Hak Pengguna

Pengguna memiliki hak untuk mengakses, memperbarui, dan mengajukan penghapusan data melalui administrator pesantren atau dengan menghubungi pengembang.

7. Penggunaan Kamera

Aplikasi dapat menggunakan kamera perangkat untuk pemindaian QR, foto profil, atau kebutuhan verifikasi operasional. Aplikasi tidak merekam video latar belakang tanpa persetujuan pengguna.

8. Perubahan Kebijakan

Kebijakan privasi ini dapat diperbarui dari waktu ke waktu mengikuti perkembangan fitur dan layanan TahfidzMU. Versi terbaru akan dipublikasikan pada halaman ini.
