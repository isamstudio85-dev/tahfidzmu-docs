Kebijakan Privasi

TahfidzMU — Aplikasi Tahfidz Mudah untuk Berbagai Hafalan

Terakhir diperbarui: 8 Juli 2026

1. Pendahuluan

TahfidzMU ("kami", "aplikasi") adalah aplikasi Tahfidz Mudah yang dikembangkan oleh iSam Studio untuk memudahkan pengelolaan hafalan secara fleksibel, modern, dan praktis. Aplikasi ini tidak hanya digunakan untuk hafalan Al-Quran, tetapi juga dapat digunakan untuk berbagai jenis hafalan lain sesuai kebutuhan lembaga. Saat ini TahfidzMU tersedia melalui aplikasi mobile, web admin untuk operasional pesantren, serta panel super admin untuk pengelolaan banyak pesantren. Kebijakan privasi ini menjelaskan bagaimana kami mengumpulkan, menggunakan, dan melindungi informasi pengguna.

2. Data yang Kami Kumpulkan

Kami mengumpulkan data berikut yang diberikan secara langsung oleh pengguna atau administrator pesantren:

  • Data Akun: Nama, username, email, dan jabatan/peran pengguna.
  • Data Santri: Nama, NIS, kelas, halaqah, nama orang tua/wali, nomor HP wali, tanggal lahir, dan foto profil.
  • Data Akademik: Riwayat setoran hafalan, riwayat tasmi, nilai, dan catatan presensi.
  • Data Pesantren: Nama pesantren, alamat, nomor telepon, dan email institusi.
  • Foto: Foto profil yang diunggah pengguna (opsional).

3. Bagaimana Kami Menggunakan Data

Data yang dikumpulkan digunakan semata-mata untuk:

  • Mengelola proses berbagai hafalan, termasuk namun tidak terbatas pada hafalan Al-Quran, setoran, murojaah, dan tasmi.
  • Menampilkan progres hafalan santri kepada musyrif, pengawas, dan orang tua.
  • Mengirim notifikasi terkait aktivitas hafalan.
  • Mengelola administrasi pesantren melalui aplikasi mobile, web admin, dan panel super admin sesuai hak akses masing-masing.
  • Autentikasi dan otorisasi pengguna.

4. Penyimpanan dan Keamanan Data

  • Semua data disimpan di Google Firebase (Firestore dan Firebase Storage) yang berlokasi di server Google dengan standar keamanan tinggi.
  • Akses data dilindungi oleh Firebase Security Rules berbasis peran (role-based access control).
  • Data setiap pesantren terisolasi — pesantren A tidak dapat mengakses data pesantren B.
  • Koneksi antara aplikasi dan server menggunakan enkripsi TLS/SSL.

5. Berbagi Data dengan Pihak Ketiga

Kami tidak menjual, menyewakan, atau membagikan data pribadi pengguna kepada pihak ketiga. Data hanya dapat diakses oleh:

  • Pengguna yang berwenang sesuai peran mereka di pesantren.
  • Pengembang aplikasi untuk keperluan pemeliharaan teknis.
  • Google Firebase sebagai penyedia infrastruktur (tunduk pada Kebijakan Privasi Google).

6. Hak Pengguna

Pengguna memiliki hak untuk:

  • Mengakses data pribadi mereka melalui aplikasi.
  • Memperbarui informasi profil mereka.
  • Meminta penghapusan data dengan menghubungi administrator pesantren atau pengembang.

7. Penggunaan Kamera

Aplikasi menggunakan kamera perangkat hanya untuk:

  • Memindai QR Code saat login.
  • Mengambil foto profil (opsional).

Kami tidak merekam video atau mengakses kamera di latar belakang.

8. Data Anak-Anak

Aplikasi ini digunakan di lingkungan pendidikan pesantren dan dapat memproses data santri di bawah usia 13 tahun. Data anak-anak hanya diinput dan dikelola oleh administrator pesantren atau musyrif yang berwenang, bukan oleh anak-anak secara langsung.

9. Perubahan Kebijakan

Kami dapat memperbarui kebijakan privasi ini dari waktu ke waktu. Perubahan akan diumumkan melalui pembaruan aplikasi. Tanggal "terakhir diperbarui" di bagian atas halaman ini akan mencerminkan perubahan terbaru.
